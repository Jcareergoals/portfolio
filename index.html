Weather Application

Get real-time weather information.

The Weather Application displays real-time weather information from any city around the world. By entering the name of a City, State, or Country of choice, users will be displayed weather information for that location. If the user happens to enter a State or Country, then weather information for that location's capital will be displayed. The information is received from a weather service API (Openweathermap.org). The technologies used to create this application include HTML5, CSS3, Ajax, JSON, jQuery, Twitter Bootstrap, CodeIgnitor (MVC), and Git.

You can preview the Weather Application here.

Pokemon Application

Gotta catch them all.

If you any need quick basic information on any pokemon characters than this awesome application will come in handy. By clicking on any of the 151 characters, information about that character will be displayed in the pokedex (the orange box). The data is pulled from an API webservice (pokeapi.co). The technologies used to build this application include HTML5, CSS3, AJAX, JSON, jQuery, and CodeIgnitor.

Poke App

Feel like poking someone?

This is a pretty cool applicatoin. Users can poke other users, see which were poked, how many times they were poked, by who they were poked by. Users can also see how many time they themselves were poked and by who as well. All you have to do is register if not registered or login to enjoy the fun.
